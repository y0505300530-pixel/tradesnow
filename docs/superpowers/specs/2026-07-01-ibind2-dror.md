# IBIND2 — Dror (port 5002)

## Port map (droplet)

| Port | Service | Account |
|------|---------|---------|
| **5000** | `ibind-oauth` | **CEO LIVE** — do not touch |
| **5001** | `ibind-oauth-paper` | Paper (DU) |
| **5002** | `ibind-oauth-dror` | **Dror LIVE** (dormant) |

## Install

```bash
cd /root/tradesnow
sudo bash scripts/setup-ibind2-dror.sh
# restart tradesnow (pm2) after .env updated
```

## Dormant guards

- `IBKR_OAUTH_PENDING=1` in `/root/ibind-oauth-dror/.env.dror` → `POST /session/start` returns **503**
- Dror `liveEngineConfig.isEnabled = 0`
- `alertPoller` = CEO only

## Wire Dror IBKR OAuth (when account opens)

1. Register OAuth app in IBKR for Dror's login
2. Update `/root/ibind-oauth-dror/.env.dror`:
   - `CONSUMER_KEY`, `ACCESS_TOKEN`, `ACCESS_TOKEN_SECRET`
   - Replace `certs/private_signature.pem` + `private_encryption.pem`
3. Set `IBKR_OAUTH_PENDING=0`
4. `systemctl restart ibind-oauth-dror`
5. `tradingAccounts.updateIbkrAccountId({ slug: 'dror', ibkrAccountId })`

## Health check

```bash
curl -s -H "Authorization: Bearer $IBIND_API_SECRET_DROR" http://127.0.0.1:5002/health
```
